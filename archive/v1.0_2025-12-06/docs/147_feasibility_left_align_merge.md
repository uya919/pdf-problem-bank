## 추가 옵션 (선택적)

### 정렬 옵션 파라미터화

더 유연한 구현을 원한다면:

```python
def merge_images_vertically(
    images: List[Image.Image],
    padding: int = 10,
    align: str = "left"  # "left", "center", "right"
) -> Optional[Image.Image]:
    ...
    for img in images:
        if align == "left":
            x_offset = 0
        elif align == "right":
            x_offset = max_width - img.width
        else:  # center
            x_offset = (max_width - img.width) // 2
        ...
```
