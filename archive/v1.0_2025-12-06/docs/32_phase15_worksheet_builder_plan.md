## 개요

기존 라벨링 시스템으로 생성된 문제 이미지들을 선택하여 A4 시험지 템플릿에 배치하고 PDF로 출력하는 기능

---

## 핵심 기능

| 기능 | 설명 |
|------|------|
| 문제 선택 | 필터/검색으로 문제 탐색 후 선택 |
| 템플릿 선택 | 4문제/6문제 등 프리셋 선택 |
| 자동 배치 | 선택한 문제를 템플릿에 자동 배치 |
| 자유 캔버스 | 필요시 드래그로 직접 배치 |
| PDF 출력 | A4 시험지 PDF 생성 |
| 커스텀 템플릿 | 사용자 정의 템플릿 생성 |

---

## 구현 단계

- [ ] 문제 목록 API (`GET /api/problems`)
